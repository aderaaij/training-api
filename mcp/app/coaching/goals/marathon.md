# Goal module: Marathon

**Entry bar:** comfortably running ~30–40 km/week *before* the plan starts.
If the athlete isn't there, build base first — a marathon plan cannot also be
the base-building.

**Plan shape (all levels):** 16–20 weeks (12 minimum on a strong base).
- Long run caps: **≤30% of weekly volume AND ~3 h**, whichever comes first —
  slower runners cap by time, not by chasing 32 km (core §5).
- **Fueling is trainable and mandatory:** carbs + fluids on every long run
  ≥90 min; full race-day nutrition rehearsed at marathon pace at least 2–3
  times.
- Marathon-pace practice matters: MP segments inside late-plan long runs.
- Taper 2–3 weeks, volume −40–60%, intensity and frequency unchanged
  (core §6). Hansons-style ~10-day tapers are a valid alternative for
  cumulative-fatigue plans.

## Beginner

Higdon-Novice shape — 16–20 weeks, **4 run days + 1 cross/strength day +
2 rest days**:
- All running easy — **no speedwork**; the long run is the only stressor.
- Long run 10 → 32 km with a stepback every 3rd week
  (e.g. 13-16-19-**13**-21-24-**16**-26…). Peak ~50–65 km/week.
- One 32 km run ~3 weeks out, then the 3-week taper.
- The goal is finishing healthy. Time goals wait for marathon #2.

## Intermediate

Pfitzinger-18/55 shape — 16–18 weeks, 5 run days, peak ~70–90 km/week:
- Signature: midweek **medium-long run 18–24 km** + weekly **LT work**
  (tempo ≥20 min growing to 6–7 km at ~15K–half pace).
- Long run 24–32 km; 2–3 of them with MP segments (e.g. final 8–13 km at
  goal pace). Optionally two 32 km runs.
- Phases: endurance → LT + endurance → race prep (5K-pace sharpening,
  optional tune-up race) → taper.

## Advanced

6 run days, peak ~90–110+ km/week. Two proven shapes — choose by recovery
capacity and psychology, and **don't mix their long-run philosophies**:
- **Pfitzinger 18/70+:** 2 quality days (LT + VO2max) + medium-long run +
  long runs 32–35 km.
- **Hansons cumulative fatigue:** long run capped at 26 km *run on
  pre-fatigued legs* (it simulates the last 26 km of the race, not the
  first), 3 SOS days/week — speed (5–10K pace) → strength (~MP minus
  6 s/km) → tempo at MP growing to 16 km — and a deliberately short
  ~10-day taper.
